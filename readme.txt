== Description ==
Mail Newsletter plugin has options to send HTML emails newsletters to subscribers. Plugin stores user information (email address) inputted by users while visiting the page or website. Mail Newsletter plugin users can view their user base, and email users. This plugin is great for those who seek an easy way to capture user information and send newsletters and automated email updates to their subscribers.

== Installation ==
1. Upload mail-newsletter.zip file in you `/wp-content/plugins/`  directory and unzip it.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Go To Mail Newsletter page and send news to your subscribers.
